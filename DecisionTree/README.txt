Instructions for Compiling and Testing:

Files included are:

	DecisionTree.py
	Driver.py
	Report.txt
	BREASTCANCER_Output.txt
	CARVALUE_Output.txt
	IRIS_output.txt
	
	
First, run DecisionTree.py file - all the helper functions are defined in this file
Then run driver.py

Here, the user is asked to input the dataset location - URL.
After this, the user is then prompted to input the Attribute values separated with space.

We used 3 datasets:
Following is the URL and Attributes of 3 datasets that can be entered as input.

IRIS 
URL: https://archive.ics.uci.edu/ml/machine-learning-databases/iris/iris.data

Attributes: SepalL SepalW PetalL PetalW Class

BREAST CANCER
URL: https://archive.ics.uci.edu/ml/machine-learning-databases/breast-cancer/breast-cancer.data

Attributes: Class age menopause tumor-size inv-nodes node-caps deg-malig breast breast-quad irradiat

CAR VALUATION:
URL: https://archive.ics.uci.edu/ml/machine-learning-databases/car/car.data

Attribute: buying maint doors persons lug_boot safety
